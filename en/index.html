behaviac

  • behaviac is a framework of the game AI development, and it also can be used as a rapid game prototype design tool.
  • behaviac supports the behavior tree(BT), finite state machine(FSM) and hierarchical task network(HTN).
  • Behaviors can be designed and debugged in the designer, exported and executed by the game.
  • The designer is a stone alone editor, which can only run on the Windows platforms.
  • The run time library is implemented with C++ and C#, and it supports all major platforms (Windows, Linux, Android, iOS, Unity etc.) and Unity.
  • The C++ version is suitable for the client and server side.
